Vue - 從零開始學 Get Started

  2020-09-05   Coding   html vue Word Count: 756 words Read Time: 2 minutes

從零開始學 Vue - 模仿篇

載入必要的 js

<code class="language-js"><!-- development version, includes helpful console warnings -->
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script></code>

宣告式渲染

el 用來找 Html 的元素，data 則是用來存放變數的地方，只要 data 內的屬性 (ex: message) 有對應到 Html 上 {{ }} 內的變數，就會被及時渲染。

驗證方法：開發人員工具 -> console -> 試著打 app.message = ‘Hello World’

<code class="language-html"><div id="app">
    {{ message }}
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            message: 'Hello Vue!'
        }
    });
</script></code>

綁定 Html 元素屬性

在 Html 的標籤內只要加上了 v-bind，Vue 就會依據其綁定的屬性去做相應的效果，如下所示，只要滑鼠移置該文字，就會出現 Vue 所控制的 Title 內容

<code class="language-html"><div id="app-2">
    <span v-bind:title="message">
        請將滑鼠移置此文字
    </span>
</div>

<script>
    var app2 = new Vue({
        el: '#app-2',
        data: {
            message: '被滑過囉!'
        }
    })
</script></code>

條件以及迴圈

Vue 除了可以綁定元素的屬性之外，也能綁定 DOM 的結構，使用的方法為 v-if，若變數為 true，則顯示 Html

這邊 Vue 處理的方式不是將 Html 多一個 Display: none 的屬性，而是將整個區塊註解，整體而言，隱密性相對更高

<code class="language-html"><div id="app-3">
    <span v-if="seen">Now you see me</span>
</div>

<script>
    var app3 = new Vue({
        el: '#app-3',
        data: {
            seen: true
        }
    })
</script></code>

除了隱藏區塊外，Vue 處理迴圈也是相當簡潔，相對於 HtmlHelper，在迴圈的處理上至少省去了三行以上 code 的撰寫，使用方法為 v-for

<code class="language-html"><div id="app-4">
    <ol>
        <li v-for="todo in todos">
            {{ todo.text }}
        </li>
  </ol>
</div>

<script>
    var app4 = new Vue({
        el: '#app-4',
            data: {
                todos: [
                { text: 'Learn JavaScript' },
                { text: 'Learn Vue' },
                { text: 'Build something awesome' }
            ]
        }
    })
</script></code>

使用者的輸入處理

對於良好的互動設計，必定要對使用者在頁面上的各種操作做一些特殊的效果，例如跳出彈出式視窗、提醒輸入格式錯誤等等，因此我們需要撰寫各種監聽事件去處理相對應的效果

對於 Vue，我們只需要在要監聽的 Html 元素上寫上 v-on:[event]，Vue js 就會去找在 method 底下相應的 function 執行

<code class="language-html"><div id="app-5">
    <p>{{ message }}</p>
    <button v-on:click="reverseMessage">Reverse Message</button>
</div>

<script>
    var app5 = new Vue({
        el: '#app-5',
        data: {
            message: 'Hello Vue.js!'
        },
        methods: {
            reverseMessage: function () {
            this.message = this.message.split('').reverse().join('')
            }
        }
    })
</script></code>

雙向繫結

Vue 同時也支援雙向綁定，在下面的例子中，只要輸入框的值改變，連同 p 標籤內的文字也會跟著改變，不需要再另外寫監聽或 change 事件 (Vue 已實作)

<code class="language-html"><div id="app-6">
    <p>{{ message }}</p>
    <input v-model="message">
</div>

<script>
    var app6 = new Vue({
        el: '#app-6',
        data: {
            message: 'Hello Vue!'
        }
    })
</script></code>
